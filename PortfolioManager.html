Markowitz Efficient Frontier¶

Describes the weight proportions of the stated stocks that yield the highest return for a given amount of risk.

In [8]:
pfolio_returns = []
pfolio_volatilities = []
num_assets = len(tickers)
log_returns = pd.DataFrame()
log_returns = np.log(myportfolio / myportfolio.shift(1))

weight_arr = []
for x in range (2000):
    weights = np.random.random(num_assets)
    weights /= np.sum(weights) #Array of random combinations of weights of stock in portfolio.
    weight_arr.append(weights)
    pfolio_returns.append(np.sum(weights * log_returns.mean()) * 250)
    pfolio_volatilities.append(np.sqrt(np.dot(weights.T,np.dot(log_returns.cov() * 250, weights))))
    
pfolio_returns = np.array(pfolio_returns) #Yearly returns for every weight combination
pfolio_volatilities = np.array(pfolio_volatilities) #Yearly volatility for every weight combination

frontier_data = pd.DataFrame({'Return': pfolio_returns, 'Volatility': pfolio_volatilities})

Euler Discretization to Simulate Stock Behaviour¶



$$ {\LARGE S_t = S_{t-1} \mathbin{\cdot} e^{((r - \frac{1}{2} \cdot stdev^2) \mathbin{\cdot} \delta_t + stdev \mathbin{\cdot} \sqrt{\delta_t} \mathbin{\cdot} Z_t)} } $$

Stock price prediction based on volatility, risk-free rate of return and a normally distributed random element.

In [10]:
stdev = log_returns.std() * (250 ** 0.5) #Yearly volatility
r = 0.025 #Risk-Free rate of return
T = 1.0 #Time period for which to predict prices in years.
t_intervals = 250 #Trading days per year.
delta_t = T / t_intervals  
iterations = 10000 #Number of simulations

mat_dict = {}
mat_opt = {}
for tick in tickers:
    Z = np.random.standard_normal((t_intervals + 1, iterations))  
    S = np.zeros_like(Z) 
    S0 = myportfolio[tick].iloc[-1]  
    S[0] = S0
    
    for t in range(1, t_intervals + 1):
        S[t] = S[t-1] * np.exp((r - 0.5 * stdev[tick] ** 2) * delta_t + stdev[tick] * delta_t ** 0.5 * Z[t])
    mat_opt[tick] = S
    mat_dict[tick] = pd.DataFrame(data=S[:,:10])

Calculating the Price of Stock Options¶

This is done by evaluating the average difference between the final stock price and strike price, and relating it to the risk-free rate of return and the time period of the option. The greater the time period, the greater the uncertainty. Thus, price of options increase with time period to help mitigate the sell side's risk.

$$ C = \frac{exp(-r \cdot T) \cdot \sum{p_i}}{iterations} $$

strike_over_today = 1.1 #i.e. Strike price is 10% over todays price
options = {}
for tick in tickers:
    options[tick] ='%.2f' %round(\
    (np.exp(-r*T)*np.sum(np.maximum(mat_opt[tick][-1]-mat_opt[tick][0][0] * strike_over_today, 0)))\
    /iterations, 2) + '$'
